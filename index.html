Daggerheart Dice — ESP32 Web Flasher

Horizontal build

This page flashes the horizontal firmware to your ESP32 (CYD) using your browser (Web Serial).

1) Add firmware

  1. Export your horizontal sketch from Arduino IDE: Sketch → Export compiled Binary.
  2. Place the generated .bin into the firmware/ folder of this repo and rename it to app.bin.
  3. Commit & push.

2) Flash it

Click Install, select your ESP32's serial port, and follow prompts. If it doesn't auto-bootload, hold BOOT when asked.

Troubleshooting

  • Use a Chromium desktop browser (Chrome/Edge/Brave).
  • Close Arduino Serial Monitor while flashing.
  • If the chip is blank, you might need to include bootloader/partitions in manifest.json (see README).
